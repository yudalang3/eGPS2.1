Problem to solve:

A module that demonstrates handy tools for biologists through dockable panel functionality.

Key features:

1. Demonstrates the usage habits of the VOICE software operation mechanism, with built-in module wizard.

2. Provides multiple bioinformatics analysis tools, including:

  • Large text generator - Generates large text files for testing
  • Simple alignment simulator - Simulates sequence alignments
  • Simple expression producer - Generates expression data
  • Group-wise statistical test - Performs statistical tests between groups
  • Advanced group-wise test - Advanced statistical analysis between groups
  • Biological pathway enrichment - Pathway enrichment analysis

Detailed Module Descriptions:

1. Large text generator

  • Parameters:
    • output.file.path: Output file location (e.g., 10GB_fastq.txt)
    • file.GiB.size: Size of the generated file in GiB (note this is not GB)
  • Description: Generates large text files of specified size for testing and data processing simulations.

2. Simple alignment simulator

  • Parameters:
    • ref.sequence: Reference DNA sequence
    • num.sequence: Total number of sequences to output, including the reference
    • include.indel: Whether to include insertions/deletions (INDELs)
    • output.file.path: Output file path
  • Description: Simulates sequence alignments, generating sequence files with mutation characteristics.

3. Simple expression producer

  • Parameters:
    • meta.info.type.number: Number of condition types (2 for control/treatment, 3 for ctrl/depletion/rescue)
    • meta.info.label: Condition labels separated by semicolons
    • output.file.path: Output file path
    • model.gene.number: Total number of genes
    • model.differentialRatio: Ratio of differentially expressed genes
    • model.baseMean: Base expression mean
    • model.dispersion: Dispersion parameter
    • model.foldChangeMin: Minimum fold change
    • model.foldChangeMax: Maximum fold change
  • Description: Generates simulated gene expression data for differential expression analysis.

4. Group-wise statistical test

  • Parameters:
    • input.expression.tsv: Input expression profile TSV file
    • condition.or.group.1: Sample names of the first condition or group
    • condition.or.group.2: Sample names of the second condition or group
    • analysis.method.name: Analysis method name
    • output.result.tsv: Output result file path
  • Description: Performs statistical tests between groups to identify significantly differentially expressed genes.

5. Advanced group-wise test

  • Parameters:
    • input.expression.tsv: Input expression profile TSV file
    • condition.or.group.1: Sample names of the first condition or group
    • condition.or.group.2: Sample names of the second condition or group
  • Description: Performs advanced group-wise statistical tests using multiple methods (e.g., limma, DESeq2, and EdgeR) for differential expression analysis.

6. Biological pathway enrichment

  • Parameters:
    • input.geneset.gmt: Input GMT format gene sets
    • input.significant.result.tsv: Input significant gene result file
    • symbol.col.name: Gene symbol column name
    • significant.col.name: Column name with p-values
    • output.file.path: Output file path
    • p.value.cutoff: P-value threshold
  • Description: Calculates pathway enrichment results for genes using classical methods such as Fisher's exact test (ORA) and GSEA.
