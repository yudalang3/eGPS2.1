解决的问题：

通过可停靠面板的功能展示为生物学家编写趁手小工具的模块。

主要功能：

1. 展示VOICE软件运行机制的使用习惯，内置了模块使用向导。

2. 提供多个生物信息学分析工具，包括：

  • Large text generator - 大型文本文件生成器
  • Simple alignment simulator - 简单序列比对模拟器
  • Simple expression producer - 简单表达数据生成器
  • Group-wise statistical test - 组间统计检验工具
  • Advanced group-wise test - 高级组间统计检验工具
  • Biological pathway enrichment - 生物通路富集分析工具

模块详细说明：

1. Large text generator（大型文本生成器）

  • 参数说明：
    • output.file.path：输出文件位置，例如 10GB_fastq.txt
    • file.GiB.size：生成文件的大小（以GiB为单位），注意这不是GB
  • 功能描述：生成指定大小的大型文本文件，用于测试和模拟数据处理。

2. Simple alignment simulator（简单序列比对模拟器）

  • 参数说明：
    • ref.sequence：参考DNA序列
    • num.sequence：输出序列的总数，包括参考序列
    • include.indel：是否包含插入/删除（INDEL）
    • output.file.path：输出文件路径
  • 功能描述：模拟序列比对，生成具有突变特征的序列文件。

3. Simple expression producer（简单表达数据生成器）

  • 参数说明：
    • meta.info.type.number：条件类型数量，2表示对照/处理，3表示对照/耗尽/救援
    • meta.info.label：条件标签，用分号分隔
    • output.file.path：输出文件路径
    • model.gene.number：基因总数
    • model.differentialRatio：差异基因比例
    • model.baseMean：基础表达均值
    • model.dispersion：离散参数
    • model.foldChangeMin：最小倍数变化
    • model.foldChangeMax：最大倍数变化
  • 功能描述：生成模拟的基因表达数据，用于差异表达分析。

4. Group-wise statistical test（组间统计检验工具）

  • 参数说明：
    • input.expression.tsv：输入的表达谱TSV文件
    • condition.or.group.1：第一个条件或组的样本名称
    • condition.or.group.2：第二个条件或组的样本名称
    • analysis.method.name：分析方法名称
    • output.result.tsv：输出结果文件路径
  • 功能描述：执行组间统计检验，识别显著差异表达的基因。

5. Advanced group-wise test（高级组间统计检验工具）

  • 参数说明：
    • input.expression.tsv：输入的表达谱TSV文件
    • condition.or.group.1：第一个条件或组的样本名称
    • condition.or.group.2：第二个条件或组的样本名称
  • 功能描述：执行高级组间统计检验，使用多种方法（如limma, DESeq2和EdgeR）进行差异表达分析。

6. Biological pathway enrichment（生物通路富集分析工具）

  • 参数说明：
    • input.geneset.gmt：输入的GMT格式基因集
    • input.significant.result.tsv：输入的显著基因结果文件
    • symbol.col.name：基因符号列名
    • significant.col.name：包含p值的列名
    • output.file.path：输出文件路径
    • p.value.cutoff：p值阈值
  • 功能描述：计算基因对应的通路富集结果，使用经典方法如ORA的Fisher精确检验和GSEA方法。

快速开始：

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

备注：

1：用户可以点击示例按钮（罗盘）查看各个例子。

2：所有参数的说明都在脚本内容的#起始的注释行中。

3：模块使用标签页组织，便于快速切换不同功能。
