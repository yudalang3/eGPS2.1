生物信息学可视化应用

这是一个基于Java Swing的生物信息学数据可视化应用程序，提供多种数据可视化功能。每个可视化功能都作为独立的JInternalFrame，可以自由拖动和排列。

功能特性

1. 多序列比对可视化 (MultipleSequenceAlignmentPanel)

  • 显示DNA/蛋白质序列的比对结果
  • 相同列用颜色标识（保守位点）
  • 支持核苷酸颜色编码：
    • A: 红色
    • T: 绿色
    • C: 蓝色
    • G: 黄色
    • 间隙(-): 灰色
  • 显示保守性指示器（*标记）

2. MA图可视化 (MAPlotPanel)

  • 用于基因表达数据的对比分析
  • M轴：log2 fold change（差异倍数）
  • A轴：平均表达量
  • 颜色编码：
    • 红点：显著上调基因
    • 蓝点：显著下调基因
    • 灰点：无显著差异基因
  • 显示显著性阈值线

3. 热图可视化 (HeatmapPanel)

  • 显示基因表达或其他数值数据的热图
  • 颜色范围：蓝色（低值）→ 白色（中值）→ 红色（高值）
  • 支持鼠标悬停显示详细数值
  • 包含颜色标尺和数值范围
  • 行列标签显示

4. 图像显示 (ImageDisplayPanel)

  • 支持多种图片格式（JPG, PNG, GIF, BMP）
  • 图片缩放功能（放大、缩小、重置）
  • 滚动查看大图片
  • 文件选择对话框
  • 状态信息显示

应用架构

主要类文件

  1. BioinformaticsApp.java - 主应用程序类
    • 创建主窗口和桌面面板
    • 管理多个独立的JInternalFrame
    • 提供菜单栏和窗口管理功能
  2. MSAInternalFrame.java - 多序列比对内部框架
    • 独立的可拖动窗口
    • 包含工具栏和操作按钮
    • 封装多序列比对面板
  3. MAPlotInternalFrame.java - MA图内部框架
    • 独立的可拖动窗口
    • 包含数据刷新和设置功能
    • 封装MA图面板
  4. HeatmapInternalFrame.java - 热图内部框架
    • 独立的可拖动窗口
    • 包含聚类和颜色设置功能
    • 封装热图面板
  5. ImageInternalFrame.java - 图像显示内部框架
    • 独立的可拖动窗口
    • 包含图像处理工具栏
    • 封装图像显示面板
  6. MultipleSequenceAlignmentPanel.java - 多序列比对面板
    • 序列对齐可视化
    • 保守性分析
    • 颜色编码显示
  7. MAPlotPanel.java - MA图面板
    • 基因表达差异分析
    • 散点图绘制
    • 统计显著性标识
  8. HeatmapPanel.java - 热图面板
    • 矩阵数据可视化
    • 交互式数据探索
    • 颜色映射
  9. ImageDisplayPanel.java - 图像显示面板
    • 图片加载和显示
    • 缩放和导航功能
    • 文件管理

示例图片来源

应用中的示例图片是通过代码动态生成的，位于 ImageDisplayPanel.java 的 loadSampleImage() 方法中（第211-250行）。该方法使用Java的 BufferedImage 和 Graphics2D API创建一个包含以下元素的示例图片：

  • 蓝色到红色的渐变背景
  • 白色圆形轮廓
  • 黄色圆点（模拟眼睛）
  • 黑色弧线（模拟嘴巴）
  • "示例图片"文字

这样设计的好处是不依赖外部图片文件，应用可以独立运行并展示图像显示功能。

使用方法

编译和运行

# 编译所有Java文件
javac *.java

# 运行应用程序
java BioinformaticsApp

操作步骤

  1. 启动应用：运行BioinformaticsApp类
  2. 显示所有窗口：点击菜单栏的"文件" → "导入数据"，会同时显示四个独立的内部框架
  3. 单独打开窗口：使用"窗口"菜单可以单独打开特定的可视化窗口
  4. 窗口操作：
    • 每个内部框架都可以独立拖动、调整大小
    • 可以最小化、最大化或关闭窗口
    • 窗口会自动排列成2x2网格
  5. 交互操作：
    • 热图：鼠标悬停查看数值
    • 图像面板：使用缩放按钮或加载自定义图片
    • 每个窗口都有专门的工具栏
    • 所有面板都支持滚动查看

设计特点

模块化设计

  • 每个可视化功能独立封装为单独的JInternalFrame
  • 每个框架包含专门的工具栏和操作按钮
  • 便于扩展和维护
  • 支持应用迁移和重用

灵活的窗口管理

  • 每个可视化组件都是独立的JInternalFrame
  • 支持自由拖动、调整大小、最小化/最大化
  • 可以单独打开或批量显示
  • 自动排列功能提供整齐的布局

用户体验

  • 直观的图形界面
  • 丰富的交互功能
  • 清晰的视觉反馈
  • 详细的状态信息
