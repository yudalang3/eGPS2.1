Bioinformatics Visualization Application

This is a Java Swing-based application for visualizing bioinformatics data, offering various data visualization features. Each visualization function is implemented as a separate JInternalFrame, which can be freely dragged and arranged.

Features

1. Multiple Sequence Alignment Visualization (MultipleSequenceAlignmentPanel)

  • Displays the alignment results of DNA/protein sequences.
  • Columns with identical residues are highlighted (conserved sites).
  • Supports nucleotide color coding:
    • A: Red
    • T: Green
    • C: Blue
    • G: Yellow
    • Gap (-): Gray
  • Displays a conservation indicator (*).

2. MA Plot Visualization (MAPlotPanel)

  • Used for comparative analysis of gene expression data.
  • M-axis: log2 fold change.
  • A-axis: Average expression level.
  • Color coding:
    • Red dots: Significantly up-regulated genes.
    • Blue dots: Significantly down-regulated genes.
    • Gray dots: Genes with no significant difference.
  • Displays significance threshold lines.

3. Heatmap Visualization (HeatmapPanel)

  • Displays a heatmap of gene expression or other numerical data.
  • Color range: Blue (low values) → White (medium values) → Red (high values).
  • Supports mouse hover to display detailed values.
  • Includes a color scale and value range.
  • Row and column labels are displayed.

4. Image Display (ImageDisplayPanel)

  • Supports various image formats (JPG, PNG, GIF, BMP).
  • Image scaling functions (zoom in, zoom out, reset).
  • Scroll to view large images.
  • File selection dialog.
  • Status information display.

Application Architecture

Main Class Files

  1. BioinformaticsApp.java - Main application class
    • Creates the main window and desktop pane.
    • Manages multiple independent JInternalFrames.
    • Provides a menu bar and window management functions.
  2. MSAInternalFrame.java - Multiple Sequence Alignment internal frame
    • A separate, draggable window.
    • Contains a toolbar and action buttons.
    • Encapsulates the multiple sequence alignment panel.
  3. MAPlotInternalFrame.java - MA Plot internal frame
    • A separate, draggable window.
    • Includes data refresh and settings functions.
    • Encapsulates the MA plot panel.
  4. HeatmapInternalFrame.java - Heatmap internal frame
    • A separate, draggable window.
    • Includes clustering and color settings functions.
    • Encapsulates the heatmap panel.
  5. ImageInternalFrame.java - Image Display internal frame
    • A separate, draggable window.
    • Contains an image processing toolbar.
    • Encapsulates the image display panel.
  6. MultipleSequenceAlignmentPanel.java - Multiple Sequence Alignment panel
    • Sequence alignment visualization.
    • Conservation analysis.
    • Color-coded display.
  7. MAPlotPanel.java - MA Plot panel
    • Gene expression difference analysis.
    • Scatter plot drawing.
    • Statistical significance indicators.
  8. HeatmapPanel.java - Heatmap panel
    • Matrix data visualization.
    • Interactive data exploration.
    • Color mapping.
  9. ImageDisplayPanel.java - Image Display panel
    • Image loading and display.
    • Zooming and navigation functions.
    • File management.

Sample Image Source

The sample image in the application is dynamically generated by code in the loadSampleImage() method of ImageDisplayPanel.java (lines 211-250). This method uses Java's BufferedImage and Graphics2D APIs to create a sample image containing the following elements:

  • A blue to red gradient background.
  • A white circular outline.
  • Yellow dots (simulating eyes).
  • A black arc (simulating a mouth).
  • The text "Sample Image".

The advantage of this design is that it does not depend on external image files, allowing the application to run independently and demonstrate its image display capabilities.

How to Use

Compile and Run

# Compile all Java files
javac *.java

# Run the application
java BioinformaticsApp

Operating Steps

  1. Start the application: Run the BioinformaticsApp class.
  2. Show all windows: Click "File" → "Import Data" in the menu bar to display four independent internal frames simultaneously.
  3. Open a single window: Use the "Window" menu to open a specific visualization window.
  4. Window operations:
    • Each internal frame can be independently dragged and resized.
    • Windows can be minimized, maximized, or closed.
    • Windows are automatically arranged in a 2x2 grid.
  5. Interactive operations:
    • Heatmap: Hover over to view numerical values.
    • Image Panel: Use the zoom buttons or load a custom image.
    • Each window has a dedicated toolbar.
    • All panels support scrolling.
