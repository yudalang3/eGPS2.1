Task to resolve

Compare the differences of the two phylogenetic trees.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Using this module only requires inputting two files in the NWK format. On the parameter panel, you can choose to include an outgroup. If an outgroup is not included, the program will directly read the data for visualization. After entering the name of the outgroup, the program will first re-root the tree based on the corresponding node of this name, and then rearrange the tree.
  2. Users can also adjust the 'Name font' to achieve a more desirable effect.
  3. Click the View tanglegram button to display the graph.

Command Line Usage

The '$libPath' is the directory where the JAVA lib is located. The following is the command line to use:

java -cp "$libPath" egps2.module.evoltree.graphics.QuickCompareTwoTreeCLI
