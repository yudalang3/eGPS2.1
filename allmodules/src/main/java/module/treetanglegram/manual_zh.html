解决的问题

本模块适合比较两棵进化树的差异。

详情信息请见：https://www.yuque.com/u21499046/egpsdoc/gfnvenvwcl96oq18

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

  1. 使用该模块只需要输入两个nwk格式的文件，在参数面板可以选择加入外群。若不加入外群则程序将会直接读入数据进行可视化，输入了外群的名字之后，程序将会先根据这个名字对应的节点重新定根，然后对树进行重新排列。
  2. 用户还可调节Name font，以得到比较理想的效果。
  3. 点击View tanglegram按钮即可显示图形。

命令行使用

$libPath为JAVA lib所在的目录，下面为使用的命令行：

java -cp "$libPath" egps2.module.evoltree.graphics.QuickCompareTwoTreeCLI
