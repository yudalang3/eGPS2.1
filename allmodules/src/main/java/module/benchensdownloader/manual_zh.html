解决的问题

Ensembl官网提供了基因组以及基因组注释（genome annotation）的下载链接https://www.ensembl.org/info/data/ftp/index.html?redirect=no 然而在很多情况下，我们都需要批量下载数据。整个过程包括：认知网站数据的组织方式，得到批量下载的链接等等。 我们使用操作平台类来处理这个问题。

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

详情见：https://www.yuque.com/u21499046/egpsdoc/hgbvy6vz95pt7u87

  1. 点击输入按钮打开数据输入对话框。
  2. 在参数输入区域查看所需输入数据。当模块提供多个示例时，可点击示例按钮切换。
  3. 用户输入所需参数。
  4. 点击执行按钮查看结果。
