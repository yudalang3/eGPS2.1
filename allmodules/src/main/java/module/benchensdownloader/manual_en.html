Task to resolve

The Ensembl official website provides download links for genomic data and genome annotations at FTP Download (https://www.ensembl.org/info/data/ftp/index.html?redirect=no ). However, in many cases, we need to download data in bulk. The entire process includes understanding the organization of the website's data and obtaining links for bulk downloads. We use an operations platform class to handle this issue.

Quick start

All user actions are straightforward. Please follow the GUI instructions.

  1. Click the import button to open the data import dialog.
  2. View the required input data in the parameter input area. When the module provides more than one example, you can click the example button to see the next.
  3. The user enters the required parameters.
  4. Click the Execute button to see the results.
