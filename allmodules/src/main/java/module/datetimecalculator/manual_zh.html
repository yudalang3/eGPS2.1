日期时间计算器

该模块为日期计算器，有两个目的：

  1. 给定date1和date2，计算两个日期之间经历的时间（天或者月）。
  2. 给定date与两个日期经历的时间，计算另一个日期。

使用方式：

请阅读软件界面上的使用说明，使用操作是十分简单的。

除示例教程之外使用场景：

场景一：我需要将2年时间，从2020-01到2022-12，平均分为4个时间段。如果解决？

解决方式：可以输入 2020-01 6，这样就会计算2020-01过6个月之后的时间。重复此过程即可得到时间段。
