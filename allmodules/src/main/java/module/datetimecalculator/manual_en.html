Datetime calculator

This module is a datetime calculator with two purposes:

  1. (1) Given two dates, date1 and date2, it calculates the time span between them (in days or months).
  2. (2) Given a date and a time span, it calculates the corresponding date after adding the specified duration.

How to Use:

Please read the instructions on the software interface, which is very straightforward to operate.

Usage Scenarios Beyond Example Tutorials:

Scenario One: I need to divide a 2-year period, from January 2020 to December 2022, into 4 equal time segments. How to do it?

Solution: You can input "2020-01 6", and it will calculate the date 6 months after January 2020. Repeat this process to get the time segments.
