Task to resolve

Find the location of some short sequences in the genome.

Example: I focus on a Motif: CTTTGTTG. I want to see, where is the exact match in the human reference genome?

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. Click the Import button on the toolbar or direct drag the text file to open the text.
  2. Click the Execute button to see the results.
