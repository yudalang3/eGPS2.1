解决的问题

本模块用来绘制序列标识图。

快速开始

所有操作都比较直观、易懂。请按照图形用户界面的指示进行操作。

详见：https://www.yuque.com/u21499046/egpsdoc/smmsm4zpwisegn5a

  1. 支持输入序列文件，fasta文件以及频率文件，具体格式可以点击Example按钮查看。可以直接将文本粘贴到文本框，也可以直接导入文件。
  2. 生成序列标识图前必须执行的操作：选择Sequence type中序列的类型，明确输入的序列是DNA，RNA还是蛋白质；选择Import data type中数据的类型，明确输入的数据格式是序列文件，fasta文件还是频率文件。
  3. 生成序列标识图前可选择的操作：选择Abscissa angle中横坐标的角度是水平还是竖直，默认为水平；选择Units中生成的序列标识图的类型是Bits还是Frequency，默认是Bits；选择Logo size中生成的序列标识图的大小，默认会根据面板的大小进行调整。
  4. 最后点击Create sequence logo按钮，生成序列标识图。

支持通过MEME-Chip的结果文件生成序列标识图

  1. 可以直接将文本粘贴到文本框，也可以直接导入文件。但是格式必须是MEME-Chip生成的meme结果文件。
  2. 生成序列标识图前必须执行的操作：选择Sequence type中序列的类型，明确输入的序列是DNA，RNA还是蛋白质
  3. 生成序列标识图前可选择的操作：选择Abscissa angle中横坐标的角度是水平还是竖直，默认为水平；选择Units中生成的序列标识图的类型是Bits还是Frequency，默认是Bits；选择Output format中输出的文件格式，默认为jpg格式；选择Output folder，最后会将文件输出到该目录的outImage目录下。
  4. 最后点击Create sequence logo按钮，生成序列标识图。会展示生成的序列标识图的前五张图片。
