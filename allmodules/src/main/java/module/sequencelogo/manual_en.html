Task to resolve

Draw sequence logo diagrams.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

  1. It supports various input formats including sequence files, FASTA files, and frequency files. For detailed formatting examples, please refer to the Example section. You can either paste your text directly into the provided text box or import files directly for processing.
  2. Operations required before generating sequence logo diagrams: Select the sequence type to specify whether the input sequence is DNA, RNA, or protein; select the data type in Import Data Type to determine whether the input data format is a sequence file, FASTA file, or frequency file.
  3. Optional operations before generating sequence logo diagrams: Select the orientation of the abscissa axis, either horizontal or vertical, with the default being horizontal; choose the type of sequence logo generated, either Bits or Frequency, with Bits being the default; select the size of the generated sequence logo, which will adjust automatically based on the panel size by default.
  4. Finally, click the "Create sequence logo" button to generate the sequence logo.

It supports generating sequence logo diagrams from MEME-Chip result files.

  1. You can either directly paste the text into the text box or import a file. However, the format must be a MEME-Chip generated MEME result file.
  2. Operations required before generating sequence logo diagrams: Select the sequence type to specify whether the input sequence is DNA, RNA, or protein.
  3. Optional operations before generating sequence logo diagrams: Select the orientation of the abscissa axis, either horizontal or vertical, with the default being horizontal; choose the type of sequence logo generated, either Bits or Frequency, with Bits being the default; select the output file format in Output Format, with the default being jpg format; choose the Output Folder, where the files will be output to the "outImage" directory within that selected folder.
  4. Finally, click the "Create sequence logo" button to generate the sequence logo. The first five generated sequence logo images will be displayed.
