A convenient module to quick operate protein sequences, process the multiple peptides sequences with pipeline.

Quick start

All user actions are straightforward. Please follow the GUI instructions.

  1. Click the import button to open the data import dialog.
  2. View the required input data in the parameter input area. When the module provides more than one example, you can click the example button to see the next.
  3. The user enters the required parameters.
  4. Click the Execute button to see the results.
