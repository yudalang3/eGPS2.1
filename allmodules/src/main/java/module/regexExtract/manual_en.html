Task to resolve

Given a text file, each line has a fixed format string. The target information is extracted from each line of string.

Quick start

Any actions are straightforward. Please follow the GUI instructions.

Application Example

By running the example file of this module, we have the following content:

Saccharomyces cerevisiae - Baker's yeast (酿酒酵母)
Caenorhabditis elegans - Nematode (秀丽隐杆线虫)
Drosophila melanogaster - Fruit fly (黑腹果蝇)

我们的想得到的结果是生成一个以Tab键分隔的文本内容如下：

Saccharomyces cerevisiae	Baker's yeast	酿酒酵母
Caenorhabditis elegans	Nematode	秀丽隐杆线虫
Drosophila melanogaster	Fruit fly	黑腹果蝇

To achieve this requirement, regular expressions would be needed if we wrote the script ourselves. This would involve using an IDE to write the script or directly in a terminal window, and the completed script is typically used only once. Such operations require a shift in thinking for researchers and are not very convenient to use.

Therefore, this module was developed, and using it only requires three steps.

  1. Generate a regular expression based on the string extraction requirements from the web-based AI model. For example, the regular expression to achieve this functionality is (.*?)\\s*-\\s*(.*?)\\s*\\((.*?)\\)
  2. Enter the remaining necessary parameters, such as the input and output file paths
  3. The third step is to run it, either through the command line or by clicking the run button on the graphical interface

Tip: The input and output file paths can be expanded or you can directly paste the string address. You can create bookmarks using the VOICM's history function for reuse in future similar requirements.

Command Line Usage

Assuming that "$libPath" is the directory where the JAVA library files are located, the following command can be used:

java -cp "$libPath" edu.sinh.beauty.unisoft.module.treenodecoll.CLI configFile

Since VOICM has the ability to run script records, the command line is actually more cumbersome to use.
