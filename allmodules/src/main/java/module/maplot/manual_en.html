The details of data transformation

Let x and y are respectively the mean of the two groups being compared. "Original Data" represent painting the scatter plot directly with x and y values。

After selecting the “Do MA plot” options, the computation details are following:

  1. “Direct”: y_new = y - x; x_new = (x + y) / 2
  2. “Log(x)”: y_new = log2(y) - log2(x); x_new = (log2(x) + log2(y)) / 2
  3. “Log(x+1)”: y_new = log2(y + 1) - log2(x + 1); x_new = ( log2(x + 1) + log2(y + 1) ) / 2
  4. “Log(x + |min(x)| + 1)”: y_new = log2(y + |min(y)| + 1) - log2(x + |min(x)| + 1);
    x_new = log2(x + |min(x)| + 1) + log2(y + |min(y)| + 1) / 2

Note: |min(x)| means get the minimal value of all x first, than apply the abs() function.
